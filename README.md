## Run code using Go tests
Run code using Go tests with verbose mode to see the answers:
```
go test ./... -v
```
or
```
./bin/test
```
Run tests for a single day:
```
go test github.com/dmowcomber/advent-of-code/2022/day5 -v
```
or
```
./bin/test 2022/day5
```
